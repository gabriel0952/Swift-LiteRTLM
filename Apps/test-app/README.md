# LiteRTLM Test App

Local Flutter app used to validate the Swift Package from this repository.

## Location

- App root: `Apps/test-app/`
- Swift Package reference: repository root via `../../../` from `ios/Runner.xcodeproj`

## Purpose

- Smoke-test engine initialization and chat flows
- Validate CPU / simulator behavior without publishing the package
- Reproduce integration issues while working on the local Swift Package
